# Mars Vista API Documentation Site

API documentation built with React, Vite, TypeScript, Tailwind CSS, and Redoc.

## Development

```bash
npm install
npm run dev
```

Visit http://localhost:5173

## Build

```bash
npm run build
```

Output in `dist/` directory.

## Structure

- `src/App.tsx` - Main component with Redoc integration
- `public/openapi.yaml` - OpenAPI specification
- Redoc theme customized with Mars Vista brand colors
